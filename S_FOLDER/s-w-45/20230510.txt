어제 배운 것

1. 인터럽트 관련한 에러가 많았음.
   - ws: 말고 wss를 https 환경에서 사용해야 함. 근데 이게 되었다가 안되서 좀 잘 모르겠다. nginx 설정 등 관련해서 건드리면 해결이 될 것 같은데, 네트워크와 운영체제 공부가 필요하구나 느꼈음.
2. 생각해보면 디자인 패턴도 찾아보고 알아서 적용한 것이다. 네트워크, 운영체제, 알고리즘을 알고 있다면 시작부터 해당 조건들을 고려하여 더 좋은 코드가 나왔을 것이다. CS 공부를 열심히 하자.
3. syncedStore와 Yjs의 Y.Doc 관련한 사용법을 대략 알았음.
   - get method를 통해 가져오며, 배열의 형태는 약간 다르다. 메서드가 몇 없음. 그렇기에 해당 메서드를 Y.array 등에서 적용을 하고 싶었다.
   - node_modules를 보니 터치가 가능하여 고칠 수 있다고 생각했으나, 내가 의도한 생각만큼 잘 되지 않았음. filter 등을 넣고 싶었다. toJSON()을 통해 일반 배열로 변경 후, map을 통해 filter 옵션에 대해 boolean 배열 형태로 만들어서 false인 값의 인덱스를 받아와서 delete 메서드를 실행하면 될 것 같음. 맵을 돌면서 삭제를 실행하는데, idx에서 이전까지 false 카운트 갯수를 늘리는 형태로. reduce인지 뭔지 써서.
